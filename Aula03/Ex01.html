Destaques no texto

Podemos realizar destaques no texto, tais como:
negrito;
texto destacado;
itálico;
texto com ênfase;
Ou ainda riscado ou sublinhado .

A diferença entre < b >, que faz o negrito, para a tag strong é que essa última dará um destaque semântico. O mesmo acontece com a tag < em >.

As alterações do tamanho e cor de fonte devem ser feitas via CSS. Essas alterações podiam ser realizadas via HTML até a versão 4, mas caíram em desuso no HTML 5.
