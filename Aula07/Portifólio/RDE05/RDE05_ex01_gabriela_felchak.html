Como utilizar hiperlinks -->

Se é usado pela tag < a>. Os hiperlinks podem ser texto, imagens, vídeos... Ainda podemos classificá-los em:

Remotos:

Referenciam páginas da web;
<a href=“URL”>Conteúdo Referenciado</a>

Internos:

Referenciam lugares dentro da própria página. Rolam a página até onde se é preciso e está marcado;
< a href="#referencia">Conteúdo Referenciado</a>
<a name=“referencia"> </a>

Locais:

Indicam outras páginas dentro do mesmo site;
<a href=“arquivo.html”>Conteúdo Referenciado</a>>

Com endereço eletrônico:

Com o nome bem autoexplicativo, apontam para um endereço eletrônico;
<a href=“mailto:email@email.com">email@email.com</a>

Com imagens:

<a href=“referencia”> <img src="imagem.jpg"> </a>
